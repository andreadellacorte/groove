# Migration: 0.11.5 → 0.12.0

## What changed

Three new memory files and a directory were introduced in the v0.12.x series that existing groove installations will not have:

- `<memory>/mistakes.md` — incident log for workflow errors (v0.12.2)
- `<memory>/promises.md` — deferred items from work sessions (v0.12.6)
- `<memory>/learned/` — long-term workflow insight directory (v0.12.x)

## Outcome

All three are created if absent. Existing files are not touched (idempotent).

## Steps

Read `memory:` from `.groove/index.md` (default: `.groove/memory/`). Let `<memory>` be that path.

### 1. Create `<memory>/learned/` directory

```bash
mkdir -p <memory>/learned
```

Report: `✓ <memory>/learned/ — created` or `✓ <memory>/learned/ — already present`

### 2. Create `<memory>/mistakes.md`

Check if `<memory>/mistakes.md` exists. If absent, write:

```markdown
# Mistakes and Lessons

## How This File Works

1. **Log it** — Record in Incident Log below when an error is identified
2. **Fix it** — Apply the fix (code, config, file, convention)
3. **Audit it** — Identify root cause; add permanent fix to `.groove/memory/learned/<topic>.md`
4. **Summarise** — Move lesson to Resolved table; delete log entry; keep file lean

## Incident Log

(Empty — no open incidents)

## Resolved

| Date | Mistake | Root Cause | Fix Applied | Learned File |
|---|---|---|---|---|
```

Report: `✓ <memory>/mistakes.md — created` or `✓ <memory>/mistakes.md — already present`

### 3. Create `<memory>/promises.md`

Check if `<memory>/promises.md` exists. If absent, write:

```markdown
# Promises

Deferred items from work sessions — things to come back to.

## Open

| # | Date | Promise | Context |
|---|---|---|---|

## Resolved

| # | Date | Promise | Resolved |
|---|---|---|---|
```

Report: `✓ <memory>/promises.md — created` or `✓ <memory>/promises.md — already present`

## Constraints

- Idempotent — if any file or directory already exists, skip and report "already present"
- Do not overwrite existing content
- Use the `memory:` value from `.groove/index.md`; default to `.groove/memory/` if key absent
