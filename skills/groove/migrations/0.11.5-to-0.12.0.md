# Migration: 0.11.5 → 0.12.0

## What changed

A new memory directory was introduced in the v0.12.x series that existing groove installations will not have:

- `<memory>/learned/` — long-term workflow insight directory (v0.12.x)

## Outcome

Directory is created if absent. Existing content is not touched (idempotent).

## Steps

Read `memory:` from `.groove/index.md` (default: `.groove/memory/`). Let `<memory>` be that path.

### 1. Create `<memory>/learned/` directory

```bash
mkdir -p <memory>/learned
```

Report: `✓ <memory>/learned/ — created` or `✓ <memory>/learned/ — already present`

## Constraints

- Idempotent — if directory already exists, skip and report "already present"
- Use the `memory:` value from `.groove/index.md`; default to `.groove/memory/` if key absent
